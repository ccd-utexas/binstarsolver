Testing
-------

Use `pytest <https://pytest.org>`_:

::

   $ git clone https://github.com/ccd-utexas/binstarsolver.git
   $ cd binstarsolver
   $ git tag --list
   $ git checkout tags/v0.1.2 # or latest tag name
   $ py.test
