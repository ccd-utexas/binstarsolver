Installation
------------

Use `Python Packaging Index <https://pypi.python.org/pypi>`_.

::

$ pip install binstarsolver

Testing
-------

Use `pytest <https://pytest.org>`_.

::

$ git clone https://github.com/ccd-utexas/binstarsolver.git
$ cd binstarsolver
$ git tag --list
$ git checkout tags/<latest_tag_name>
$ py.test
